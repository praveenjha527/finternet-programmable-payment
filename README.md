# Finternet Payment Gateway Documentation

Welcome to the Finternet Payment Gateway API documentation. This documentation provides everything you need to integrate programmable payment infrastructure into your application.

## What is Finternet?

Finternet is a trustless payment infrastructure platform that enables programmable money movement between blockchain and traditional financial systems. Built for merchants who need secure, transparent, and automated payment processing.
